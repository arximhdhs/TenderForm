# Tender Landing Page

A simple GitHub-hosted landing page for validating demand for the **Tender app**.

This project is designed to act as a lightweight marketing website where visitors can:
- learn what the app is about
- see the value proposition
- join a waitlist or express interest
- submit their email for updates

The site is primarily static and hosted on **GitHub Pages**, with **Firebase** used for basic backend operations such as storing email submissions.

## Purpose

The main goal of this project is to validate product demand before building the full Tender app.

This landing page helps:
- measure interest in the product
- collect early user emails
- test messaging and positioning
- build an initial waitlist

## Tech Stack

- **HTML / CSS / JavaScript**
- **GitHub Pages** for hosting
- **Firebase** for basic backend features
  - email collection
  - simple form handling
  - optional analytics or future lightweight features

## Features

- Responsive landing page
- Product overview and value proposition
- Email capture form
- Firebase integration for storing submitted emails
- GitHub Pages deployment
- Simple and low-cost setup for MVP validation
